Tipos primitivos

Em Javascritp, há dois tipos de dados possíveis: primitivos (passados por valor) e objetos (passados por referência).

Os tipos primitivos são:

  • String
  • Number
  • Boolean
  • Undefined
  • Null
  • Symbol (ES2015)
  • BigInt (ES2020)

Os tipos primitivos podem ser usados para armazenar dados simples e primitivos.
